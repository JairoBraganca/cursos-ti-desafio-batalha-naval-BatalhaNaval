# Batalha Naval

Este é um programa simples em C que simula um tabuleiro de 10x10 posições. O programa inicializa o tabuleiro com água (representada por 0) e posiciona três tipos diferentes de formas geométricas em posições específicas.

## Descrição das Formas Geométricas

- **0**: Água (posição vazia)
- **1**: Forma tipo 1 - Cruz (formato de cruz centrada)
- **2**: Forma tipo 2 - Diamante (formato de diamante)
- **3**: Forma tipo 3 - Triângulo (formato triangular)
## Saída do Programa

O programa imprime o tabuleiro no formato de uma grade 10x10, com:
- Colunas rotuladas de A a J
- Linhas numeradas de 1 a 10
- Cada célula mostrando o valor correspondente (0, 1, 2 ou 3)


Nota: A saída exata pode variar dependendo da implementação específica, mas representa o posicionamento dos navios conforme descrito no código.

## Posicionamento dos Navios

- **Navio Tipo 3 (Triângulo)**: Centrado na posição (6,6) [linha 7, coluna G], formando um triângulo de altura 3.
- **Navio Tipo 1 (Cruz)**: Centrado na posição (6,2) [linha 7, coluna C], formando uma cruz.
- **Navio Tipo 2 (Diamante)**: Centrado na posição (3,6) [linha 4, coluna G], com tamanho 2, formando um diamante.
